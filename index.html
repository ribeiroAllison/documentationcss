CSS Selectors


Types of Selectors

Type Target Description Syntax Example
Type Elements of certain Type Will target all elements of a certain type Name of the type followed by curly brackets ({}) h1{}
Class Defined Class Will target all elements identified with the given class Is called by a dot (.) before the name of the class .example{}
ID Unique ID Will target the element identified with the unique ID called Is called by a hashtag (#) before the name of the ID #example{}
Groups All elements within selectors list Will select different element and apply the same rules to them Is called by declaring different elements separated by a comma (,) h1, h2{}
Chain Elements that meet all defined criteria Will target an element if and only it meets ALL the criteria defined by the selectors One element followes by another without blank space h1.example{}
All All elements Will target all elements of a file An asterisk followed by curly brackets *{}
Descendant Elements descendants to other elements Targets an element within another one that is it's parent Parent element follow by its children separated by a single space div p{}

Important Concepts

Concept Details
HTML attributes with multiple values Some HTML attributes can have multiple attribute values. Multiple attribute values are separated by a space between each attribute.
Selector Specificity Specificity is a ranking system that is used when there are multiple conflicting property values that point to the same element. When determining which rule to apply, the selector with the highest specificity wins out. The most specific selector type is the ID selector, followed by class selectors, followed by type selectors.
